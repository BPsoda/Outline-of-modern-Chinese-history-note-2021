- 鸦片战争前后中国的社会性质？

    - “师夷长技以制夷”的主张和早期维新思想：
      1. 林则徐“睁眼看世界第一人”
      2. 魏源编纂《海国图志》，提出师夷长技以制夷。主张学习外国先进的军事和科学技术，以期富国强兵。
      3. 19世纪70年代以后，王韬、薛福成、马建忠、郑观应等人不仅主张学习西方的科学技术，还要求吸纳西方的政治、经济学说。

    - 人民的反侵略斗争：

      2. 太平天国运动后期，太平军多次重创英、法侵略军和外国侵略者指挥的洋枪队“常胜军”“常捷军”；
      3. 台湾人民抗日斗争；
      4. 义和团抵御八国联军。
    - 爱国官兵的反侵略斗争：
      1. 1859年英法联军进攻大沽炮台，中国守军击伤击沉敌舰多艘；
      2. 1884年中法战争，法国海军入侵台湾被台湾事务大臣刘铭传的守军击退；
      3. 中越边境的镇南关，七旬老将军冯子材身先士卒，大败法军，取得镇南关大捷。
      4. 甲午战争致远舰邓世昌、经远舰林永升。


## 第二章 对国家出路的探索
### 太平天国农民战争
- 性质：农民起义
  - 《天朝田亩制度》和《资政新篇》的内容、特点
    -  《天朝田亩制度》  
      - 内容：土地期均匀分配——以解决土地问题为中心的比较完整的社会改革方案。  
      - 特点：  
        进步性：否定了封建地主土地所有制，是对以往农民战争“均贫富”思想的发展超越；  
        局限性：其社会形式受到小生产者的眼界局限，是一种自给自足的闭塞经济；是一种空想的不切实际的形式。
    - 《资政新篇》
      - 内容：政治上加强中央集权，学习西方法治；经济上发展近代事业，提倡资本主义雇佣劳动制；文化上设立新闻馆、学校、医院等社会福利事业；主张同外国平等交往。
      - 特点：是一种系统性的发展资本主义的方案，受限于环境未能得以实现。

## 第三章 辛亥革命与君主专制制度的终结
- 性质：资产阶级民主革命
- 意义
    推翻了清王朝反动统治，沉重打击中外反动势力；
    结束了封建君主专制，建立了资产阶级共和政府；
    带给了人民思想上的解放；
    促使社会经济、思想习惯和社会风俗发生了新的积极变化；
    推动了亚洲各民族解放运动的高涨；
- 为什么说它是失败的？又为什么说它是成功的？  
    - 失败：袁世凯窃取了辛亥革命的果实，建立了代表买办资产阶级和大地主利益的北洋军阀反动政权，在政治、经济和文化上对辛亥革命进行了反攻倒算；  
    - 成功：推翻了清王朝的封建统治，结束了两千余年的封建专制制度；使得民主共和思想在中国广为流传。

## 第四章 开天辟地的大事变（重点）
### 新文化运动
- 基本内容：
    是资产阶级民主主义新文化反对封建主义旧文化的斗争；  
    基本口号是民主与科学。提倡民主，反对专制；提倡科学，反对盲从迷信；  
    批判孔学，但不反对全部的中国传统文化。
- 作用：  
    掀起了思想解放的潮流。

### 五四运动
- 性质：新民主主义的开端
- 过程：
    - 导火索：巴黎和会外交失败；
    - 开端：1919.5.4北京学生三千余人在天安门集会，举行示威游行，遭到北洋政府的镇压；
    - 发展：1919.6.5上海七万工人罢工，推动商人罢市、学生罢课，反帝爱国运动扩展到二十余省区，百余城市；
    - 结果：北洋政府于6.10罢免了亲日派官僚，6.28没有出席巴黎和会签字仪式。
- 意义
    - 表现了反帝反封建的彻底性，使中国人民的反帝反封建斗争上升到了一个新的水平；
    - 是一次真正的群众运动；
    - 促进了马克思主义在中国的传播及其与中国工人运动的结合；

### 学习思考（P128)
- 中国的先进分子为什么和怎样选择了马克思主义？
  - 为什么：（可自由发挥）  
    历史条件  
    1. 中国的先进分子选择马克思主义是近代以来先进中国人向西方探索救国救民真理历史发展的必然结果。农民阶级、洋务派、维新派、资产阶级革命派探索国家出路的努力先后失败，先进分子向西方寻求真理的努力一再碰壁，开始对西方文明产生怀疑和持保留态度。 
    2. 北洋军阀统治黑暗，中国的出路问题再一次摆到中国人民面前。 
    3. 五四运动前后中国工人阶级的壮大及其斗争为中国的先进分子选择马克思主义提供了阶级基础和实践需求。 
    4. 新文化运动的兴起，思想启蒙运动的结果，为马克思主义在中国的传播创造了前提条件。
    5. 第一次世界大战和十月革命的影响，推动了中国的先进分子思想方向的转变。第一次世界大战充分暴露了资本主义制度的内在矛盾，中国人对资本主义方案产生了怀疑;俄国十月革命给陷入彷徨、苦闷中的中国人昭示了新的理想目标和建国方案，使先进分子看到了斗争的新希望。   

    思想运动
    1. 在李大钊等人的影响和当时形势的推动下，一批爱国的进步青年，尤其是那些具有初步共产主义思想的知识分子，经过各自的摸索，逐步划清了资产阶级民主主义和无产阶级社会主义、科学社会主义和其他社会主义流派的界限，走上了马克思主义的道路。  

  - 怎样选择：
    1. 重视对马克思主义基本理论的学习，明确地同第二国际的社会民主主义划清界限。
    2. 注意从中国实际出发，学习、运用马克思主义理论。
    3. 开始提出知识分子应当同劳动人民相结合的思想。

- 为什么说中国共产党的成立是开天辟地的大事变？
  1. 共产党的成立，中国革命不仅有了新的领导阶级和革命政党，而且有了科学的指导思想和革命纲领，<u>深刻改变了近代以后中国革命和中华民族发展的进程和方向</u>。
  2. 它的成立时中国革命有了可信赖的组织者和领导者，标志着中国革命终于有了一个坚强的领导核心。
  3. 中国共产党成立之后，中国革命不仅有了明确的奋斗目标，而且有了新的革命方法和革命精神，<u>深刻改变了近代以后中国人民和中华民族的前途和命运</u>。
  4. <u>中国共产党的初心和使命，就是为中国人民谋幸福，为中华民族谋复兴。</u>为了实现这个初心和使命，中国共产党始终把通过革命实现社会主义、共产主义作为自己的奋斗目标。
  5. 中国共产党的成立，沟通了中国革命和世界革命的联系，<u>深刻改变了近代以后世界发展的趋势和格局</u>。
  6. 中国共产党的成立，是社会主义主张再世界上人口最多的东方国家扎根落地的开端，是中国共产党人影响世界和为人类作出贡献的开端。
- 什么是中国共产党的初心和使命？为什么必须“不忘初心，牢记使命”？
- 中国共产党成立后，中国革命呈现了哪些新面貌？  
  1. 第一次提出了反帝反封建的民主革命的纲领，为中国人民指出了明确的斗争目标。1922年7月召开的中国共产党第二次全国代表大会提出党的最高纲领是实现社会主义、共产主义。党再当前阶段的纲领应该是：打倒军阀；推翻国际帝国主义的压迫；统一中国为真正的民主共和国。
  2. 发动工农群众展开革命斗争。在中国共产党的领导、组织、推动下，中国掀起了第一个工人运动的高潮。同时，中国共产党也开始从事发动农民的工作。这种新式的农民运动，在中国共产党成立之前是不曾由过的。
  3. 实行国共合作，在合作中发挥主导作用，掀起大革命高潮。国民党“一大”通过的宣言对三民主义做出了新的解释，确定了联俄、联共、扶助农工的三大革命政策。不仅促进了工农运动的恢复和发展，而且开展了以推翻北洋军阀统治为目标的北伐战争。在这场革命中，中国共产党起着独特的、不可替代的作用。没有中国共产党，就不会有这场大革命。
