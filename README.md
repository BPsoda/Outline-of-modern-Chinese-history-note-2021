# 近代史纲要复习笔记
## 综述、第一章
- 鸦片战争前后中国的社会性质？  
  - 之前：封建社会。
  - 之后：半殖民地半封建社会。
- 为了救亡图存做了哪些努力？
  - 向西方学习
    - “师夷长技以制夷”的主张和早期维新思想：
      1. 林则徐“睁眼看世界第一人”
      2. 魏源编纂《海国图志》，提出师夷长技以制夷。主张学习外国先进的军事和科学技术，以期富国强兵。
      3. 19世纪70年代以后，王韬、薛福成、马建忠、郑观应等人不仅主张学习西方的科学技术，还要求吸纳西方的政治、经济学说。
  - 反侵略
    - 人民的反侵略斗争：
      1. 广州三元里人民的抗英斗争是中国近代史上第一次大规模的反侵略武装斗争；
      2. 太平天国运动后期，太平军多次重创英、法侵略军和外国侵略者指挥的洋枪队“常胜军”“常捷军”；
      3. 台湾人民抗日斗争；
      4. 义和团抵御八国联军。
    - 爱国官兵的反侵略斗争：
      1. 1859年英法联军进攻大沽炮台，中国守军击伤击沉敌舰多艘；
      2. 1884年中法战争，法国海军入侵台湾被台湾事务大臣刘铭传的守军击退；
      3. 中越边境的镇南关，七旬老将军冯子材身先士卒，大败法军，取得镇南关大捷。
      4. 甲午战争致远舰邓世昌、经远舰林永升。


## 第二章 对国家出路的探索
### 太平天国农民战争
- 性质：农民起义
  - 《天朝田亩制度》和《资政新篇》的内容、特点
    -  《天朝田亩制度》  
      - 内容：土地期均匀分配——以解决土地问题为中心的比较完整的社会改革方案。  
      - 特点：  
        进步性：否定了封建地主土地所有制，是对以往农民战争“均贫富”思想的发展超越；  
        局限性：其社会形式受到小生产者的眼界局限，是一种自给自足的闭塞经济；是一种空想的不切实际的形式。
    - 《资政新篇》
      - 内容：政治上加强中央集权，学习西方法治；经济上发展近代事业，提倡资本主义雇佣劳动制；文化上设立新闻馆、学校、医院等社会福利事业；主张同外国平等交往。
      - 特点：是一种系统性的发展资本主义的方案，受限于环境未能得以实现。
### 洋务运动
- 性质：封建统治阶级的自救运动
- 内容、事业
  兴办近代企业
  建立新式海陆军
  创办新学堂，派遣留学生

## 第三章 辛亥革命与君主专制制度的终结
- 性质：资产阶级民主革命
- 意义
    推翻了清王朝反动统治，沉重打击中外反动势力；
    结束了封建君主专制，建立了资产阶级共和政府；
    带给了人民思想上的解放；
    促使社会经济、思想习惯和社会风俗发生了新的积极变化；
    推动了亚洲各民族解放运动的高涨；
- 为什么说它是失败的？又为什么说它是成功的？  
    - 失败：袁世凯窃取了辛亥革命的果实，建立了代表买办资产阶级和大地主利益的北洋军阀反动政权，在政治、经济和文化上对辛亥革命进行了反攻倒算；  
    - 成功：推翻了清王朝的封建统治，结束了两千余年的封建专制制度；使得民主共和思想在中国广为流传。

## 第四章 开天辟地的大事变（重点）
### 新文化运动
- 基本内容：
    是资产阶级民主主义新文化反对封建主义旧文化的斗争；  
    基本口号是民主与科学。提倡民主，反对专制；提倡科学，反对盲从迷信；  
    批判孔学，但不反对全部的中国传统文化。
- 作用：  
    掀起了思想解放的潮流。

### 五四运动
- 性质：新民主主义的开端
- 过程：
    - 导火索：巴黎和会外交失败；
    - 开端：1919.5.4北京学生三千余人在天安门集会，举行示威游行，遭到北洋政府的镇压；
    - 发展：1919.6.5上海七万工人罢工，推动商人罢市、学生罢课，反帝爱国运动扩展到二十余省区，百余城市；
    - 结果：北洋政府于6.10罢免了亲日派官僚，6.28没有出席巴黎和会签字仪式。
- 意义
    - 表现了反帝反封建的彻底性，使中国人民的反帝反封建斗争上升到了一个新的水平；
    - 是一次真正的群众运动；
    - 促进了马克思主义在中国的传播及其与中国工人运动的结合；

### <u>中国共产党的诞生</u>
- 性质：以马克思主义理论为指导的工人阶级政党；
- 过程
  - 早期组织：中国革命的客观需求  
    - 中国工人阶级政党最早在上海建立，1920年8月陈独秀、李汉俊、李达等组织发起，名为共产党，陈独秀为书记
    - 同年10月，李大钊、张国焘等在北京成立共产党的早期组织
    - 1920年秋至1921年春，董必武、陈潭秋、包惠僧等在武汉，毛泽东、何叔衡等在长沙，王尽美、邓恩铭等在济南，谭平山、谭植棠等在广州，都成立了共产党的早期组织。法国、日本留学生也成立了这样的组织。
- 意义  
  1. 中国共产党的成立，是一个“开天辟地的大事变”。
  2. 中国共产党的初心和使命，就是为中国人民谋幸福，为中华民族谋复兴。
  3. 深刻改变了近代以后中国革命和中华民族发展的进程和方向。
  4. 深刻改变了近代以后中国人民和中华民族的前途和命运。
  5. 深刻改变了近代以后世界发展的趋势和格局。
- 以及其他所有相关的

### 国共合作
- 国共合作和中国共产党的关系  
  1. 中国共产党起着独特、不可替代的作用。没有共产党就没有这场大革命。
  2. 大革命是在由共产党提出的反对帝国主义、反对军阀的政治口号下进行的。
  3. 中国共产党推动了革命军的建立、在军中进行卓有成效的政治工作；在战中身先士卒，起先锋表率作用。
  4. 发动五卅爱国运动，打击日英法帝国主义；领导上海工人三次武装起义，有力配合北伐军东进。

- 中国共产党成立后，中国革命呈现了哪些新面貌？  
  1. 第一次提出了反帝反封建的民主革命的纲领，为中国人民指出了明确的斗争目标。1922年7月召开的中国共产党第二次全国代表大会提出党的最高纲领是实现社会主义、共产主义。党再当前阶段的纲领应该是：打倒军阀；推翻国际帝国主义的压迫；统一中国为真正的民主共和国。

  3. 实行国共合作，在合作中发挥主导作用，掀起大革命高潮。国民党“一大”通过的宣言对三民主义做出了新的解释，确定了联俄、联共、扶助农工的三大革命政策。不仅促进了工农运动的恢复和发展，而且开展了以推翻北洋军阀统治为目标的北伐战争。在这场革命中，中国共产党起着独特的、不可替代的作用。没有中国共产党，就不会有这场大革命。

## 第五章 中国革命的新道路（重点）
- 社会性质
  - 大革命失败后，中国半殖民地半封建的性质没有改变。国民党实行文化专制主义，维护帝国主义、封建主义、官僚资本主义。
- 过程
  - 1927年8月7日，汉口会议。
    - 内容：第一，总结大革命失败教训，纠正了陈独秀右倾机会主义，选出了一瞿秋白为首的新的临时中央政治局；第二，确定了土地革命和武装反抗国民党反动派的总方针，把发动农民举行秋收起义为当前党最主要任务。  
    - 意义：是中国共产党在政治上大大前进了一步，开始了从大革命失败到土地战争兴起的转折。
  - 走农村包围城市、武装夺取政权的道路
    - 八七会议，初步提出相机占领某个县或几个县、建立革命政权、实行武装割据的思想。
    - 毛泽东1928《中国的红色政权为什么能够存在？》《井冈山的斗争》指出以农业为主要经济的中国，以军事发展暴动是一种特征；科学地阐述了共产党领导的土地革命、武装斗争与根据地建设这三者之间的辩证统一关系。1930《星星之火可以燎原》。
    - 意义：标志着中国化的马克思主义即毛泽东思想的初步形成。这是马克思主义在中国的创造性运用发展。
  - 反围剿战争与土地战争
    - 1930年10月起，3次反围剿；1932年底，第四次反围剿胜利
    - 1931年2月，毛泽东和邓子恢制定了土地革命中的阶级路线和土地分配方法。
    - 胜利原因：根本的原因在于中国共产党紧紧地依靠了农民，领导农民进行了土地制度的革命。
  - 中华苏维埃共和国
    - 1931年11月，江西瑞金
    - 内容：工民兵代表大会制度；注重发展文化教育事业，提高工农群众的文化水平。加强廉洁从政，反对贪污腐化。
  - 长征、遵义会议、红军会师
    - 1934年10月中旬，撤离根据地，开始长征。1935年3、4月，红四方面军出发。同年11月红二、六方面军出发。
    - 1935年1月遵义会议，开始确立以毛泽东为主要代表的马克思主义的正确路线在党中央的领导地位，从而在极其危急的情况下挽救了中国共产党、挽救了中国工农红军、挽救了中国革命，成为中国共产党历史上一个生死攸关的转折点。为党和革命事业转危为安、不断打开新局面提供了重要保障。
    - 1936年10月，红二、四方面军先后同红一方面军在甘肃会宁、静宁将台堡会师。长征胜利结束。
    - 意义：
      1. 宣告了国民党反动派消灭中国共产党和红军的图谋彻底失败，宣告了中国共产党和红军肩负着民族希望胜利实现了北上抗日的战略转移，实现了中国共产党和中国革命事业从挫折走向胜利的伟大转折。
      2. 是一步伟大的革命英雄主义史诗，宣告了中国共产党和其领导的人民军队是一支不可战胜的力量。
      3. 长征精神是中国共产党及其领导的人民军队革命风范的生动反映，是中华民族自强不息的民族品格的集中体现，是以爱国主义为核心的民族精神的最高体现。长征精神为中国革命不断从胜利走向胜利提供了强大的精神动力。

## 第六章 中华民族的抗日战争（重点）
- 性质：是中国人民反抗日本帝国主义侵略的民族解放战争
- 过程
  - 1931九一八事变，1935华北事变
  - 1927年7月7日卢沟桥事变
  - 停止内战，一致对外
    - 1935年12月一二九运动，促进了中华民族的觉醒，标志着中国人民救亡图存运动新高潮的到来。
    - 1935年12月，瓦窑堡会议，提出了在抗日的条件下与民族资产阶级重建统一战线的新政策。
    - 1936年12月，西安事变。
    - 1937年8月，军队改编。9月22日，国民党发表了《中共中央为公布国共合作宣言》。
  - 国民党正面战场
    - 战略防御阶段（1937年7月卢沟桥事变到1938年广州、武汉失守）
    - 战略相持阶段：趋于消极，实行保守的收缩政策，保存实力。
  - 全面抗战的路线和持久战的方针
    - 1927年8月洛川会议，指定抗日救国十大纲领，关键在于使已发动的抗战成为全面的全民族抗争
    - 1938年5月至6月，毛泽东发表《论持久战》的讲演。
  - 敌后战场的开辟于游击战争的发展
    - 地位与作用：在抗日战争的初期和中期，游击战被提到战略的地位，具有全局性的意义，成为人民军队作战的主要形式。
  - 坚持抗战、团结、进步的方针
- 意义  
  世界：
  - 中国人民抗日战争是世界反法西斯战争的重要组成部分，中国战场是世界反法西斯战争的东方主战场。中国人民的抗日战争开展时间最早、持续时间最长。
  - 抗日战争的胜利，与世界所有爱好和平与正义的国家和人民、国际组织及各种反法西斯力量的同情和支持是分不开的。
  民族：
  - 为中华民族由近代以来陷入深重危机走向伟大复兴确立了历史转折点。
  - 彻底粉碎了日本帝国主义殖民奴役中国的图谋。
  - 促进了中华民族的大团结，形成了伟大的抗日精神。
  - 对世界各国夺取反法西斯战争的胜利、维护世界和平的事业产生了重大影响。
  - 开辟了中华民族实现伟大复兴的光明前景。
- 共产党的角色与土地政策
  - 共产党是抗日战争的中流砥柱
