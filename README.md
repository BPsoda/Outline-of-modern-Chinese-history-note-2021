      1. 林则徐“睁眼看世界第一人”

      3. 19世纪70年代以后，王韬、薛福成、马建忠、郑观应等人不仅主张学习西方的科学技术，还要求吸纳西方的政治、经济学说。

      1. 广州三元里人民的抗英斗争是中国近代史上第一次大规模的反侵略武装斗争；

      3. 台湾人民抗日斗争；
      4. 义和团抵御八国联军。
    - 爱国官兵的反侵略斗争：
      1. 1859年英法联军进攻大沽炮台，中国守军击伤击沉敌舰多艘；
      2. 1884年中法战争，法国海军入侵台湾被台湾事务大臣刘铭传的守军击退；
      3. 中越边境的镇南关，七旬老将军冯子材身先士卒，大败法军，取得镇南关大捷。
      4. 甲午战争致远舰邓世昌、经远舰林永升。


## 第二章 对国家出路的探索
### 太平天国农民战争
- 性质：农民起义
- 《天朝田亩制度》和《资政新篇》的内容、特点
    -《天朝田亩制度》
      -内容：土地期均匀分配——以解决土地问题为中心的比较完整的社会改革方案。
      -特点：
        进步性：否定了封建地主土地所有制，是对以往农民战争“均贫富”思想的发展超越；
        局限性：其社会形式受到小生产者的眼界局限，是一种自给自足的闭塞经济；是一种空想的不切实际的形式。
    -《资政新篇》
      -内容：政治上加强中央集权，学习西方法治；经济上发展近代事业，提倡资本主义雇佣劳动制；文化上设立新闻馆、学校、医院等社会福利事业；主张同外国平等交往。
      -特点：是一种系统性的发展资本主义的方案，受限于环境未能得以实现。

## 第三章 辛亥革命与君主专制制度的终结
- 性质：资产阶级民主革命
- 意义
    推翻了清王朝反动统治，沉重打击中外反动势力；
    结束了封建君主专制，建立了资产阶级共和政府；
    带给了人民思想上的解放；
    促使社会经济、思想习惯和社会风俗发生了新的积极变化；
    推动了亚洲各民族解放运动的高涨；
- 为什么说它是失败的？又为什么说它是成功的？  
    - 失败：袁世凯窃取了辛亥革命的果实，建立了代表买办资产阶级和大地主利益的北洋军阀反动政权，在政治、经济和文化上对辛亥革命进行了反攻倒算；  
    - 成功：推翻了清王朝的封建统治，结束了两千余年的封建专制制度；使得民主共和思想在中国广为流传。

## 第四章 开天辟地的大事变（重点）
### 新文化运动
- 基本内容：
    是资产阶级民主主义新文化反对封建主义旧文化的斗争；  
    基本口号是民主与科学。提倡民主，反对专制；提倡科学，反对盲从迷信；  
    批判孔学，但不反对全部的中国传统文化。
- 作用：  
    掀起了思想解放的潮流。

### 五四运动
- 性质：新民主主义的开端
- 过程：
    - 导火索：巴黎和会外交失败；
    - 开端：1919.5.4北京学生三千余人在天安门集会，举行示威游行，遭到北洋政府的镇压；
    - 发展：1919.6.5上海七万工人罢工，推动商人罢市、学生罢课，反帝爱国运动扩展到二十余省区，百余城市；
    - 结果：北洋政府于6.10罢免了亲日派官僚，6.28没有出席巴黎和会签字仪式。
- 意义
    - 表现了反帝反封建的彻底性，使中国人民的反帝反封建斗争上升到了一个新的水平；
    - 是一次真正的群众运动；
    - 促进了马克思主义在中国的传播及其与中国工人运动的结合；
